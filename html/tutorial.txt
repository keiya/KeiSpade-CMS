*Wikiの記法
記法は書きやすく分かりやすいはてな記法をベースとしており、はてな記法を記述することでページが作成できます。

**見出し
「*」（半角アスタリスク）で始まる行は自動的に見出しになります。たとえば、
>||
**今日のできごと
||<
と書くと、
>>
**今日のできごと
<<
のように表示されます。アスタリスクの数で、見出しの深さが決まります。

**カテゴリー（タグ）
新規作成・編集のページでTitle: の欄に[ ]で囲んだタグを記入すると、そのカテゴリーに振り分けられ、あとで参照するときにわかりやすくなります。

**リスト
「-」（半角ハイフン・マイナス）や「+」（半角プラス）で始まる行は自動的にリストになります。また、「-」「+」の数を増やすと、その数だけ深い階層になります。たとえば、
>||
-ぶどう
--巨峰
--マスカット
-もも
||<
と書くと、

>>
-ぶどう
--巨峰
--マスカット
-もも
<<
のように表示されます。

また、「+」を使うと数字つきのリストになります。
>||
+ぶどう
+もも
++すもも
||<
と書くと、
>>
+ぶどう
+もも
++すもも
<<
のように表示されます。
通常のリストの下の階層に数字つきリストを作ることもできますが、同じ階層内に通常のリストと数字つきリストを混在させることはできません。

**リンク
直接URLを書くとURLにリンクされます。
***リンクしたくない場合
リンクしたくない場合は、
>||
[]http://www.example.com[]
||<
と書きます。

***リンクタイトルを設定する場合
>||
[http://www.example.com:title=example]
||<
と書くと、
>>
[http://www.example.com:title=example]
<<
のようにリンクが張られます。

***InterWiki Link（Wiki内リンク）
>||
[[TopPage]]
||<
と二重ブラケットで囲むと、Wiki内リンクとなり、かこったページへリンクが張られます。
ページ名の直後に「:title=」を書き、続けてタイトルを書くとリンクタイトルも設定できます。

**定義リスト
「:」（半角コロン）で始まり、「:タイトル:説明」という形式で書かれた行は自動的に定義（DL）リストになります。
>||
:京都府:京都市
:滋賀県:大津市
:三重県:津市
||<
と書くと、
>>
:京都府:京都市
:滋賀県:大津市
:三重県:津市
<<
のように表示されます。
なお、「:」を含む文字列をタイトル部分に書くことはできないため、以下のように記述してください。
URLやid記法の場合は、その部分を半角角カッコ [ 〜 ] で囲めは正しく解釈されます。
それ以外の場合は、全角文字「：」または文字実体参照「:」をお使いになるようお願いいたします。

**マルチメディアファイル
***画像
>||
[http://www.example.com/image.jpg:image]
||<
と記述すると、そのURLが画像を示している場合、その画像を直接表示させることができます。「:image」の後ろに「:h**」や「:w**」(**部分は整数)と続けて記述すると、表示するサイズをピクセル単位で調節することができます。

***ビデオ
>||
[http://www.example.com/video.mp4:video]
||<
と記述すると、そのURLがビデオファイルを示している場合、HTML5の機能を使ってそのビデオを直接表示させることができます。「:image」の後ろに「:h**」や「:w**」(**部分は整数)と続けて記述すると、表示するサイズをピクセル単位で調節することができます。

***オーディオ
>||
[http://www.example.com/audio.mp3:audio]
||<
と記述すると、そのURLがオーディオファイルを示している場合、HTML5の機能を使ってそのオーディオを直接再生させることができます。

また、画像以外のページのURLである場合は、そのページのスクリーンショットを表示させることができます。スクリーンショットは通常サイズのほか、「:large」と続けて記述することでラージサイズを、「:small」と続けて記述することでスモールサイズを表示させることができます。

また、「:image=」に続けて画像のURLを指定することで、任意の画像に対してリンクを設定することができます。

**表組み（テーブル）
半角の縦棒「|」（キーボードの「￥」とShiftキー同時押し）で区切ることで、日記の文章中で簡単にテーブルを記述できます。頭に「*」をつけた文字列は項目名となります。
>||
|*名前|*色|*個数|
|りんご|赤|1|
|みかん|だいだい|2|
||<
と記述すると、
>>
|*名前|*色|*個数|
|りんご|赤|1|
|みかん|だいだい|2|
<<
のように表示されます。

**整形済みテキストブロック（pre）
「>|」だけの行と、「|<」が行末に含まれる行ではさむことで、改行やスペースをそのまま表示する整形済みテキストブロックを作ることができます。この中では自動リンクするタイプのはてな記法やHTMLタグ、自動キーワードリンクが有効になります。
自動キーワードリンクやHTMLタグを展開せず、HTMLのソースやプログラムのコードをそのまま記述したい場合はスーパーpre記法をお使いください。
>||
>|
整形済みテキストです。
|<
||<
や
>||
>|
整形済みテキストです。|<
||<
と書くと、
>>
>|
整形済みテキストです。
|<
<<
と表示されます。

**スーパーpre記法
編集ページから入力したテキストがそのまま表示される整形済みテキストブロックを作成するには、テキストを「>||」と「||<」だけの行で囲みます。
この記法を用いると、HTMLタグや各種省略記法は無効となり、例えばプログラムのソースコードを貼り付ける場合などに利用できます。

*Wikiの設定
細かな設定を行う場合は、dat/kspade.confを作成、編集してください。
#はコメントになります。
要素とオプションがそれぞれスペースで区切られ、以下のような設定ファイルとなります。
>||
SiteName [site name]
SiteDescription [site description]
UploaderName [uploader script filename]
||<

**SiteName
Wikiタイトルです。大きく上に表示される他、HTMLのタイトルにもなります。

**SiteDescription
Wikiの解説文です。

**UploaderName
uploader.plのファイル名を変更したときは、この設定項目を追加してください。
